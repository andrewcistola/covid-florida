To replicate raw data file:

Web Address = https://fdoh.maps.arcgis.com/apps/webappviewer/index.html?id=55e34acc061c4224b9cfd54278e22927

Select = "Case Data Query"

Select:
	County = -empty-
	Gender = -empty-
	Jurisdiction = -empty-
	Hospitalized  = (blank)
	Case is on = (blank)
	Result layer name = Case Data Query _Query result

Press "Apply"

Wait

Press "..." ("Featyre Actions")

Press down arrow icon multiple times to load all results

Wait

Select "Export to CSV File"
